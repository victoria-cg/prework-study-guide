HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • It's best to use semantic HTML elements when possible for documentation clarity.

CSS

  • CSS provides style to a website.
  • CSS styles HTML elements based on their corresponding attributes that link to the stylesheet.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Stylesheets must be linked to the head of the HTML document in order to apply to the HTML code

Git

  • Git is a version control software that allows you to create commits in a repository such as github.
  • Git commands can be given via the command line interface, or CLI, and must start with the word 'git'.
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript adds interactive functions to a website.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
